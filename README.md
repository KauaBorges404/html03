# Sistema de Inscrição de Curso

Este é um projeto simples de uma página web que simula um formulário de inscrição para cursos. Ele foi desenvolvido em HTML5 puro como parte de uma atividade acadêmica/prática para exercitar o uso de diferentes elementos de formulário.

## 🚀 Tecnologias Utilizadas

* **HTML5:** Estruturação da página e criação dos campos do formulário.

## 📋 Funcionalidades

O formulário coleta as seguintes informações do usuário:
* **Nome:** Campo de texto simples.
* **Email:** Validação básica de formato de e-mail.
* **Idade:** Campo numérico.
* **Data de Nascimento:** Seletor de data (calendário).
* **Senha:** Campo de texto oculto para segurança.
* **Turno Preferencial:** Seleção de opção única (Radio Button) entre Manhã, Tarde ou Noite.
* **Curso de Interesse:** Seleção de múltiplas opções (Checkbox) entre Desenvolvimento Web, Desenvolvedor Java e Python.

## 🔧 Como Executar o Projeto

1. Baixe ou copie o código do arquivo `index.html`.
2. Abra a pasta onde o arquivo foi salvo.
3. Dê um duplo clique no arquivo `index.html` para abri-lo diretamente no seu navegador de preferência (Google Chrome, Firefox, Edge, etc.).

## 📝 Observações
* O formulário atualmente não possui estilização (CSS) ou comportamento lógico de envio (JavaScript/Backend), focando estritamente na estrutura dos elementos HTML.
